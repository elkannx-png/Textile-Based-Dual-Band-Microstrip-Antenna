# Textile-Based-Dual-Band-Microstrip-Antenna
Designed and fabricated a flexible jeans-based dual-band microstrip antenna operating at 2.45 GHz and 5.8 GHz for wearable wireless communication applications, achieving high gain, low SAR values of approximately -45 dB and -40 dB, and low reflection coefficients using CST Studio Suite.
